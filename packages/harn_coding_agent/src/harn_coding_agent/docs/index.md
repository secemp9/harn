# Harn Documentation

Harn is a minimal terminal coding harness. It is designed to stay small at the core while being extended through Python extensions, skills, prompt templates, themes, and harn packages.

## Quick start

Install Harn with pip:

```bash
pip install harn
```

Or with uv:

```bash
uv add harn
```

Then run it in a project directory:

```bash
harn
```

Authenticate with `/login` for subscription providers, or set an API key such as `ANTHROPIC_API_KEY` before starting harn.

For the full first-run flow, see [Quickstart](quickstart.md).
